## Getting Started with react-code-tag

Usage:

`npm i react-code-tag` 
##### or
`yarn add react-code-tag`

```xml
import CodeTag from "react-code-tag"

...
render () {
   return <CodeTag code={/* your code */} />
}
```

Contribute:
####
1. ```git clone https://github.com/SVladikO/react-code-tag.git```
####
2. ```cd react-code-tag```
####
3. ```npm install```
####
4. ```npm run start```
