## 1. Install:

`$ npm i react-code-tag` 
###### or
`$ yarn add react-code-tag`

## 2. Usage 
```xml
import CodeTag from "react-code-tag"

...
render () {
   return <CodeTag code={/* your code */} />
}
```

## 3. You can customize colors too:
```xml
const style = {
  codeColor: '#11fffb',
  borderColor: '#ff0303',
  backgroundColor: '#4f67d1',
  rowNumberColor: '#000',
}

render () {
   return <CodeTag code={/* your code */} style={style}/>
}
```
